Prerequisites
=============

Install xmlsec prerequisites.
Check https://github.com/mehcode/python-xmlsec


Install
=======

.. code:: console

    $ pip install aeat-web-services


Develop
=======

.. code:: console

    $ pip install -e .
    $ pip install -r requirements_test.txt


Testing
=======

.. code:: console

    $ pip install tox
    $ tox
